Personal Projects

UI/UX Projects

In designing UI/UX, I start by understanding the users’ needs and the goals of the system to ensure the interface is both functional and user-friendly. I use Figma as my main design tool because it supports real-time collaboration, responsive design, and quick prototyping. My design approach focuses on clarity, consistency, and making sure the user journey feels smooth and intuitive across all screens. I pay close attention to layout, spacing, and visual hierarchy to make interactions simple and engaging. Regular feedback is a key part of my process, helping me improve and adjust the design based on user and team input. This ensures the final product meets both user expectations and project requirements.

Mobile Applications

In mobile application development, I have solid experience working with both .NET MAUI and Flutter, allowing me to build cross-platform apps efficiently. I use MAUI for its seamless integration with C# and the .NET ecosystem, which is ideal for enterprise-level and native-like applications. On the other hand, I use Flutter when the project calls for highly customized UI and fast development cycles. My workflow includes planning user flows, creating clean and responsive interfaces, and ensuring smooth performance across devices. I focus on delivering apps that are both visually appealing and easy to use. With my experience in these frameworks, I can adapt to different project needs and deliver high-quality mobile solutions.
