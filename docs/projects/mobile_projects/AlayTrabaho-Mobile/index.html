AlayTrabaho Mobile

Date Started: March 2025

Tools Used:

Source/Reference: https://github.com/jkBicare/AlayTrabaho

Description:

AlayTrabaho is a job posting application available on both mobile and web platforms, designed to connect job seekers with recruiters in a simple and efficient way. Users can browse job listings posted by recruiters, making it easier to find opportunities that match their skills and interests. I created the design for ALayTrabaho during our internship bootcamp, where we were tasked to develop a pet project using the companyâ€™s tech stack. The goal was to help us practice, explore, and gain real experience with the tools being used in a professional setting. I designed the UI in Figma with a focus on clarity and user accessibility across both platforms. For the development, we used Flutter to build the mobile app and the T3 stack (Next.js, TypeScript, Tailwind, etc.) for the web version. The application uses an API to manage and share content data between both platforms seamlessly. The interface follows a modern and responsive design, making job browsing intuitive for users regardless of device. ALayTrabaho allowed me to apply both front-end and back-end knowledge in a real-world project. It stands as a reflection of my growth during the internship, blending design and development into a practical solution for job seekers.
