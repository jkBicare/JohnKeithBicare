BMI Calculator BEN10 Theme

Date Started: October 2024

Tools Used:

Source/Reference: https://github.com/jkBicare/BMICalc

Description:

The Ben 10-themed BMI Calculator is a mobile application I created during college for a weekly mobile development activity. Like my previous project, it was built using .NET MAUI and follows the MVVM architecture. The app calculates the user's Body Mass Index based on their height and weight inputs. Originally, I designed a Pokémon-themed version, but I felt unsure and unsatisfied with how it turned out. Because of that, I decided to redesign the project using a Ben 10 theme, which I felt more confident and inspired about. I used Figma to create a fresh UI layout along with custom assets like themed buttons, icons, and backgrounds inspired by Ben 10’s color palette and futuristic vibe. This new version gave me a better sense of direction and visual consistency. Although it didn’t target a specific audience, it was submitted to our professor as part of our class requirement. Through this project, I got to apply what I learned in UI/UX design, data binding, and MVVM structure. It helped me better understand how to redesign and improve based on feedback and self-reflection. The app delivers BMI results with easy-to-understand visuals and a fun, animated theme. Overall, it reflects my willingness to improve, experiment with design themes, and grow as a developer.
