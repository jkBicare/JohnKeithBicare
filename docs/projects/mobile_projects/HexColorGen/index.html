Hex Color Generator

Date Started: October 2024

Tools Used:

Source/Reference: https://github.com/jkBicare/HexColorGenerator

Description:

Hex Color Generator is a mobile application I developed during college for a weekly Mobile Development activity. It allows users to generate hex color codes by adjusting three separate RGB scale bars—red, green, and blue. One unique feature is a gem element that changes its color in real-time based on the selected RGB values. The app also includes a convenient option to copy the generated hex code to the clipboard. I designed the entire interface in Figma, creating all the necessary assets including medieval-style buttons and visuals. The chosen medieval theme gave the app a unique aesthetic, blending classic design elements with modern functionality. I built the app using .NET MAUI and followed the MVVM structure for better maintainability and organization. Though it wasn’t aimed at a specific audience, it was created for submission to our professor as part of our coursework. The design focuses on user interaction and visual feedback through the color-changing gem. It also helped me practice combining UI/UX design with practical functionality. This project allowed me to explore color manipulation, real-time UI updates, and user input handling. Overall, it’s a creative and interactive tool that reflects my technical and artistic growth as a student developer.
