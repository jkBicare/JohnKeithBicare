Tiptap

Date Started: October 2024

Tools Used:

Source/Reference: https://github.com/jkBicare/tiptap

Description:

TipTap is a mobile application I developed during college as part of our weekly Mobile Development activity. It is designed to help users calculate tips quickly and easily. The app includes features such as setting the number of people, choosing a tip percentage, entering a custom tip amount, and using a scale bar to adjust the tip. I chose a galaxy theme for the UI to make the app more visually appealing and unique. All design assets, including buttons and layouts, were created in Figma. The app was developed using .NET MAUI and follows the MVVM architecture for better structure and code organization. I made sure each feature works smoothly and intuitively for users. TipTap not only calculates tips but also allows for splitting the bill accurately. This project helped me apply both UI/UX principles and technical development concepts. Overall, TipTap reflects my creativity and growth as a student developer.
