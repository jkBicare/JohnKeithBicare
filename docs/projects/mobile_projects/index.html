Mobile Applications Projects

BMI Calculator (Pokemon)

The Pokémon-themed BMI Calculator is a .NET MAUI mobile app I developed in college for a weekly mobile development activity, featuring custom-designed assets in Figma and built using the MVVM structure to calculate BMI with a fun and engaging Pokémon-inspired interface.

BMI Calculator (BEN10)

The Ben 10-themed BMI Calculator is a .NET MAUI mobile app I redesigned during college using Figma and the MVVM pattern after feeling unsatisfied with my original Pokémon version, aiming for a more cohesive and exciting user experience.

Tiptap

TipTap is a galaxy-themed mobile tip calculator I developed in college using .NET MAUI and MVVM, featuring options for tip percentage, custom amounts, number of people, and a scale bar—designed in Figma for a class project.

Hex Color Generator

Hex Color Generator is a medieval-themed mobile app I created in college using .NET MAUI and MVVM, featuring adjustable RGB scale bars, a dynamic gem that changes color based on values, and the ability to copy the generated hex code designed in Figma for a weekly activity.

OneApp Lab

OneApp Lab is a multifunctional mobile app prototype I designed in Figma, combining tools like a calculator, converter, and to-do list into one clean and user-friendly interface, built as both a productivity solution and a Flutter learning project.

AlayTrabaho Mobile

Alaytrabaho is a job posting application available on both mobile and web, designed in Figma and built using Flutter and the T3 stack, allowing users to browse job listings from recruiters while serving as a hands-on learning project during our internship bootcamp.
