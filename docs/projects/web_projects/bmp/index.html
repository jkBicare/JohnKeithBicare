BuildMaster Pro

Date Started: January 2025

Tools Used:

Source/Reference: https://github.com/PhilipJohnSegarra/BuildMasterPro

Description:

BuildMaster Pro is a project management system designed specifically for the construction industry, developed as part of our thesis project. We created this system to address the challenges faced by JJ&EE Construction Supply and Development Services, who were relying on multiple tools to manage their projects. By combining these tools into one unified platform, BuildMaster Pro aims to streamline their project management process and improve efficiency. I designed the UI and mockup using Figma to help our team visualize the structure of the system and to present a clear, professional concept to our client. The system itself was built using ASP.NET Blazor, allowing us to develop both the functionality and appearance of the platform. For the design, I chose a minimalist layout with a violet and yellow color scheme—violet representing creativity and ambition, and yellow symbolizing clarity and energy, which aligns with the system’s goal of simplifying complex project workflows. Overall, BuildMaster Pro is our solution to bring structure, clarity, and modern project management to the construction field.
