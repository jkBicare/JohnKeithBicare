Eventify Web

Date Started: April 2024

Tools Used:

Source/Reference: https://www.figma.com/design/f9VOgJwg8xMpE8wQKnNEAA/IntSys-Eventify?node-id=0-1&t=oMw67S4kNyKfEAGq-1

Description:

Eventify is a global contest management system that aims to revolutionize the way contests and competition are organized and executed. Eventify allows organizers to create and customize contest with ease, providing options for defining competition categories, entry requirements, and judging criteria. Participants can register direclty to the eventify Platform, providing essential information and submitting necessary documentation effortlessly. Eventify provides the tools and support that you need to bring a fair judgement.
