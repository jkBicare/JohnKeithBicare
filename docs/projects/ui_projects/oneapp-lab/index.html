One App Lab Mobile

Date Started: April 2025

Tools Used:

Source/Reference: https://www.figma.com/design/U302xwNrE9TCZkweY1pXlE/OneApp?node-id=0-1&t=PqPLBuVO6QzAHvyq-1

Description:

OneApp Lab is a multifunctional mobile application I designed during my free time while doing my internship at RAFIMFI. It brings together a variety of everyday tools—such as a calculator, unit converter, to-do list, and more—into a single, convenient app. The idea behind OneApp Lab is to provide users with essential features they commonly use, all in one clean and simple interface. I created the mockup using Figma, making sure the design is user-friendly and well-organized for easy navigation. This project also serves as my practice ground for learning and exploring Flutter, as the system will be implemented using that framework. It allowed me to apply both UI/UX design principles and mobile development concepts in a single project. The minimalist layout ensures the app is lightweight and accessible, ideal for daily use. I chose a modern and neutral color scheme to keep the focus on functionality while maintaining a polished look. OneApp Lab is not only a helpful productivity tool for users, but also a meaningful step in my journey as a mobile developer. It reflects my passion for creating useful, practical solutions while continuously learning and improving my skills.
