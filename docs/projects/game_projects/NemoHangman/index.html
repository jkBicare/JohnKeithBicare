Nemo Hangman

Date Started: October 2024

Tools Used:

Source/Reference: https://github.com/jkBicare/NemoHangman

Description:

Hangman is a mobile game application I developed during college as part of our weekly Mobile Development activity. Itâ€™s based on the classic Hangman game, where players guess a hidden word by selecting letters. I gave it a unique twist by using a Nemo theme instead of the traditional stick figure. In this version, Nemo lives in an aquarium, and each wrong guess causes the water level to decrease. If the player loses, Nemo runs out of water and dies, but if they win, Nemo survives. This added a more emotional and engaging element to the game. I designed the interface and all assets, including the aquarium, buttons, and Nemo graphics, using Figma. The game was built using .NET MAUI and follows the MVVM pattern to separate logic and UI efficiently. Although it was not targeted at a specific audience, it was submitted to our professor as part of a graded activity. The gameplay is interactive, easy to understand, and visually appealing. This project helped me explore game logic, UI animations, and conditional interactions. Overall, it reflects my creativity in reimagining a classic game while applying development concepts in a fun and memorable way.
