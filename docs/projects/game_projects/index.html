Nemo Hangman

Hangman is a Nemo-themed mobile game I created in college using .NET MAUI and MVVM, where each wrong guess drains water from Nemo’s aquarium, putting his life at risk, and winning saves him—featuring custom assets and UI designed in Figma for a weekly class activity.
